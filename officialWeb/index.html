Everything You Need

Production-grade kiosk system optimized for Raspberry Pi 5

🧠

AI Detection Engine

Hybrid YOLOv8n + motion detection with configurable confidence threshold. Person confirmed in under 200ms.

🎬

Smart Video Playback

mpv with hardware acceleration. Fade-in/out transitions. Supports any video format. 1080p @ 30fps.

🔄

Finite State Machine

Strict 10-state FSM. No race conditions. No double triggers. Event-driven architecture.

📊

Modern UI Dashboard

PySide6 touch-friendly interface. Dark/Light theme. Live camera preview. Real-time logs.

🔌

Plugin System

Extend with custom detection engines, playback backends, or UI modules. Drop-in Python plugins.

🛡️

Watchdog & Recovery

Auto-restart on freeze (10s timeout). Camera auto-reconnect. systemd service with fail2ban.

Technical Specifications

Hardware

  • Board: Raspberry Pi 5 (8GB)
  • Camera: USB Webcam (any)
  • Display: HDMI (1080p+)
  • Storage: 32GB+ SD Card

Software

  • OS: PiOS Bookworm 64-bit
  • Python: 3.11+
  • UI: PySide6 (Qt6)
  • AI: YOLOv8n + OpenCV

Performance

  • Startup: < 15 seconds
  • Detection: < 200ms latency
  • Idle CPU: < 15%
  • Video: 1080p @ 30 FPS

Detection

  • Mode: Hybrid (YOLO + Motion)
  • Confidence: 55% threshold
  • Cooldown: 8 seconds
  • Frame Skip: Every 3rd frame

Install in 60 Seconds

No manual configuration required. Just plug and play.

1

Clone the Repository

git clone https://github.com/WeAreTheArtMakers/modInteractive.git
cd modInteractive
2

Connect Hardware

Plug your USB webcam into the Pi 5. Connect HDMI display. Insert power.

3

Run Installer

sudo ./install.sh

Installs dependencies, downloads YOLOv8 model, sets up systemd service, and starts the kiosk.

💡

That's it! Your kiosk is now running. Walk in front of the camera to test the welcome video.
